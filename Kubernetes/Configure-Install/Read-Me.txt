Ingress:
===========================================
Tested on Docker-Desktop on WSL2 Linux Terminal 
===========================================
for ingress installation it is much better to install it using Helm.

to install the Helm, open file = Install - Helm.txt and execute these commands on Linux Terminal.

once Helm installation is done, check by running the command ->  Helm

now to install the ingress, open file = Install - Ingress.txt and execute these commands on Linux Terminal.
